# Айдой

Қазақ тілді дауыстық AI көмекші. Алиса сияқты, бірақ қазақша.

## Мүмкіндіктер

- **Уақыт** — қазіргі уақыт пен күнді қазақша айтады
- **Ауа райы** — кез-келген қаладың ауа райын хабарлайды
- **Музыка** — жергілікті музыканы ойнатады, кідіртеді, тоқтатады
- **Еске салғыш** — уақытты белгілеп, дауыспен ескертеді
- **AI сұхбат** — Qwen арқылы кез-келген сұраққа жауап береді

## Орнату

```bash
# Тәуелділіктерді орнату
make install

# Ollama іске қосу (бөлек терминалда)
make ollama-start

# Qwen моделін жүктеу (бір рет)
make ollama-pull
```

`.env` файлын ашып, `WEATHER_API_KEY` мәнін қосыңыз ([openweathermap.org](https://openweathermap.org) — тегін).

## Іске қосу

```bash
make run          # мәтін режимі
make run-voice    # дауыс режимі (микрофон + дауыс шығысы)
make run-both     # мәтін кірісі + дауыс шығысы
```

## Мысалдар

```
Сіз: Айдой, қазір сағат нешеде?
Айдой: Қазір сағат 14:30, 2026 жылдың 26 наурызы, бейсенбі.

Сіз: Алматыда ауа райы қалай?
Айдой: Алматы қаласының ауа райы: Ашық аспан. Температура 18°C...

Сіз: 10 минуттан кейін кездесуге еске сал
Айдой: Жарайды, 10 минуттан кейін ескертемін: кездесуге

Сіз: музыка ойна
Айдой: Ойнатылуда: Abai - Kozimnin Karasy
```

## Конфигурация

| Айнымалы | Сипат | Әдепкі |
|---|---|---|
| `OLLAMA_BASE_URL` | Ollama сервер мекенжайы | `http://localhost:11434` |
| `OLLAMA_MODEL` | Модель атауы | `qwen2.5:7b` |
| `WEATHER_API_KEY` | OpenWeatherMap API кілті | — |
| `DEFAULT_CITY` | Әдепкі қала | `Алматы` |
| `MUSIC_DIR` | Музыка қалтасы | `~/Music` |
| `LOG_LEVEL` | Лог деңгейі | `INFO` |

## Талаптар

- Python 3.12+
- [Ollama](https://ollama.com)
- Микрофон (дауыс режимі үшін)
